## What is Mayavi?

Mayavi is an open-source, interactive 3D scientific data visualization library for Python. It integrates seamlessly with NumPy and SciPy, enabling researchers and scientists to create publication-quality visualizations of volumetric data, isosurfaces, vector fields, and point clouds.

Infrastructure teams building scientific visualization environments benefit from Mayavi's Python-based architecture and open-source flexibility. System administrators appreciate the cross-platform support for Windows, macOS, and Linux, and the ability to embed visualizations in Qt or wxPython applications.

## Key Features

| Feature | Description |
|---------|-------------|
| **3D Visualization** | Advanced 3D rendering of volumetric data, isosurfaces, vector fields, and point clouds |
| **NumPy Integration** | Direct work with NumPy arrays for efficient data handling |
| **Interactive Tools** | Data manipulation, camera control, and clipping planes |
| **GUI Integration** | Embed visualizations in Qt or wxPython applications |
| **Animation** | Create animations of time-varying data |
| **Scripting** | Python-based scripting for automation |
| **Open Source** | BSD-licensed, free for commercial and personal use |
